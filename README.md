# Manwha Kicks Reviews

A website dedicated to reviewing manhwa, sharing recommendations, and connecting fans of the genre.

## Project Description

Manwha Kicks is a capstone project for CS2100. The site features:
- Trending manhwa reviews and ratings
- Genre-based browsing (Action, Romance, Fantasy, Horror)
- A community section for user ratings and discussions
- Products page with shopping cart powered by Snipcart
- Contact form with JavaScript validation
- jQuery image slideshow on the home page
